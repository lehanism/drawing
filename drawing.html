Valid Commands

Enter the commands below and hit the enter key or press the enter button. Don't type the speech marks!

clear 'colour': Set the whole canvas to the given colour

circle 'x' 'y' 'radius' 'colour': Draw an unfilled circle with a border of the given colour

rectangle 'x' 'y' 'w' 'h' 'colour': Draw an unfilled rectangle with a border of the given colour

line 'x1' 'y1' 'x2' 'y2' 'colour': Draw a line with the given colour
